# GestureControlPC 快速入门指南

## 系统要求

- **操作系统**: Windows 10/11
- **Python**: 3.9+
- **摄像头**: USB摄像头或笔记本内置摄像头
- **内存**: 8GB+

## 安装步骤

### 1. 克隆项目

```bash
git clone https://github.com/your-repo/GestureControlPC.git
cd GestureControlPC
```

### 2. 创建虚拟环境

```bash
python -m venv venv
venv\Scripts\activate  # Windows
```

### 3. 安装依赖

```bash
pip install -r requirements.txt
```

### 4. 运行程序

```bash
python main.py
```

## 基本操作

### 启动/停止
- 点击 **▶ 开始** 按钮启动手势识别
- 或按 **空格键** 快速切换

### 全屏模式
- 点击 **⛶ 全屏** 按钮
- 或按 **F11** 键
- 按 **ESC** 退出全屏

### 设置
- 点击 **⚙ 设置** 打开配置面板
- 可调整透明度、灵敏度、手势映射等

## 默认手势

| 手势 | 动作 | 说明 |
|------|------|------|
| 👆 单指 | 空格键 | 播放/暂停 |
| ✌️ 双指 | 上一首 | 媒体控制 |
| 🤟 三指 | 下一首 | 媒体控制 |
| 🖐️ 五指张开 | Alt+Tab | 切换应用 |
| ✊ 握拳 | 左键点击 | 鼠标操作 |
| 👌 OK | Ctrl+C | 复制 |
| ❤️ 比心 | 星空心心 | 触发特效 |

## 安全区域

默认只有屏幕中央区域（20%-80%）的手势才会触发动作，防止误触。

可在设置中调整或禁用。

## 自定义配置

编辑 `config/` 目录下的 JSON 文件：

- `settings.json` - 全局设置
- `gestures.json` - 手势动作映射
- `effects.json` - 特效参数
- `calibration.json` - 摄像头校准

## 常见问题

### 摄像头无法打开
1. 检查摄像头是否被其他程序占用
2. 确认系统已授权摄像头访问权限
3. 尝试更换摄像头设备ID（设置 -> 摄像头）

### 手势识别不准确
1. 确保光线充足
2. 保持手部在画面中央
3. 减少背景杂乱
4. 调整检测置信度（设置 -> 识别）

### 特效不显示
1. 确认特效已启用（设置 -> 特效）
2. 检查是否正确做出比心手势
3. 点击 **✨ 特效** 按钮测试
